# ЛингвоСкрин

Веб-приложение для автоматической проверки рекламных текстов на наличие потенциальных нарушений Федерального закона «О рекламе» (ФЗ-38).

## Функциональность

Сервис анализирует введённый текст и выявляет два типа нарушений:

### 🔴 Недостоверная или недобросовестная реклама
Суперлативы, бездоказательные утверждения, ценовые манипуляции:
- «лучший», «№ 1», «лидер рынка», «единственный», «уникальный»
- «клинически доказано», «рекомендован врачами», «научно доказано»
- «бесплатно», «даром», «экономия», «без переплат»
- и более 90 других паттернов

### 🟡 Манипуляции пользователем
Приёмы давления, создание искусственной срочности, эксклюзивность:
- «только сегодня», «последний шанс», «успейте», «торопитесь»
- «вы заслуживаете лучшего», «не рискуйте здоровьем близких»
- «специальное предложение», «персональная скидка», «вы выбраны»
- и более 30 других паттернов

## Результат анализа

- **Нарушений нет** — текст рекомендуется к публикации без доработки
- **Есть нарушения** — выводится список найденных фраз с рекомендацией отредактировать текст, а также визуальная подсветка фрагментов в исходном тексте

## Технологии

- **React** — UI-библиотека
- **Vite** — сборщик
- **Tailwind CSS** — стилизация
- **TypeScript** — типизация

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
